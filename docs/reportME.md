## 🗄️ Реализованные сущности

### 1. Таблица `teachers` (Преподаватели)
Хранит информацию о преподавателях музыкальной школы:
- ФИО
- Контактный телефон
- Email
- Дата приёма на работу

**Количество записей:** 3 преподавателя

### 2. Таблица `instruments` (Инструменты)
Справочник музыкальных инструментов:
- Название инструмента
- Описание

**Количество записей:** 4 инструмента (фортепиано, скрипка, гитара, флейта)

### 3. Таблица `teacher_instruments` (Специализация преподавателей)
Промежуточная таблица связи «многие-ко-многим» между преподавателями и инструментами. Показывает, какие инструменты ведёт каждый преподаватель.

### 4. Таблица `students` (Ученики)
Информация об учениках:
- ФИО
- Телефон и email
- Дата рождения
- Дата зачисления в школу

**Количество записей:** 3 ученика

### 5. Таблица `lessons` (Уроки/Записи)
Основная таблица расписания:
- Ссылки на ученика, преподавателя и инструмент
- Дата и время занятия
- Длительность в минутах
- Статус (запланировано, проведено, отменено, пропущено)
- Дата создания записи

**Количество записей:** 10 уроков

### 6. Таблица `homework` (Домашние задания)
Учёт домашних заданий:
- Ссылки на ученика, преподавателя и инструмент
- Дата выдачи
- Срок сдачи (дедлайн)
- Описание задания
- Статус выполнения

**Количество записей:** 3 домашних задания

## 🔍 Реализованные запросы

### Запрос 1: Преподаватели без свободных окон
**Цель:** Найти преподавателей, у которых нет свободных слотов в ближайшие 2 дня.
**Метод:** GROUP BY с агрегатной функцией COUNT и условием HAVING.
**Результат:** Показывает загруженность преподавателя Соколова (8 уроков за 2 дня).

### Запрос 2: Расписание ученика
**Цель:** Вывести все уроки конкретного ученика с информацией о преподавателях и инструментах.
**Метод:** INNER JOIN четырёх таблиц.
**Результат:** Расписание ученика Иванова Максима (4 урока).

### Запрос 3: Нагрузка преподавателей
**Цель:** Посчитать общую нагрузку каждого преподавателя по инструментам.
**Метод:** GROUP BY с функциями COUNT и SUM.
**Результат:** Показывает, что преподаватель Соколов ведёт 8 уроков по фортепиано (480 минут).

### Запрос 4: Просроченные домашние задания
**Цель:** Найти учеников, не сдавших ДЗ в срок.
**Метод:** JOIN с условием сравнения дат и функцией DATEDIFF.
**Результат:** Пустой результат (все ДЗ ещё не просрочены относительно текущей даты сервера).

## 🛠️ Технические особенности

### Используемая СУБД
MySQL (хостинг Beget, версия 8.0+)

### Кодировка
utf8mb4 — для корректной поддержки кириллицы

### Ограничения целостности
- Первичные ключи (PRIMARY KEY) с автоинкрементом
- Внешние ключи (FOREIGN KEY) для связей между таблицами
- Уникальные индексы (UNIQUE) для предотвращения дублирования
- Проверочные ограничения (CHECK) для валидации данных

### Индексы
Созданы индексы для ускорения поиска:
- `idx_lessons_datetime` — по дате урока
- `idx_lessons_student` — по ученику
- `idx_lessons_teacher` — по преподавателю
- `idx_homework_deadline` — по сроку сдачи ДЗ
