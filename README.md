## 📋 Описание предметной области
Система учёта записи учеников на уроки музыки в частной школе. Каждый преподаватель ведёт один или несколько инструментов. Ученик может записываться к разным преподавателям, но не более одного урока в день по одному инструменту. Реализована таблица домашних заданий с датами выдачи и дедлайнами.

## 🛠 Стек технологий
- СУБД: MySQL 5.7
- Кодировка: utf8mb4
- Инструменты управления: phpMyAdmin (хостинг Beget), MySQL Workbench
- Версионирование: Git + GitHub

##  Структура репозитория
В корне проекта находится файл README с описанием работы. Папка sql содержит основной скрипт создания и наполнения схемы базы данных schema.sql. Папка queries хранит четыре отдельных файла с учебными запросами, каждый из которых решает конкретную бизнес-задачу согласно варианту. В папке docs расположен отчёт о выполнении работы и подпапка images со скриншотами результатов выполнения запросов. Папка diagrams предназначена для размещения ER-диаграммы. Файл .gitignore исключает из репозитория системные, временные и конфиденциальные файлы.

## 🚀 Развёртывание базы данных
1. Склонируйте репозиторий:
2. Откройте phpMyAdmin или консоль MySQL.
3. Создайте новую базу данных с кодировкой utf8mb4.
4. Импортируйте файл sql/schema.sql через вкладку «Импорт» или выполните его содержимое.
5. База готова к работе. Проверьте таблицы через SHOW TABLES.
